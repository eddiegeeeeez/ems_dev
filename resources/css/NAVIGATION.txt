COMMON TASKS
============

Add new component style:
  1. Create: resources/css/components/my-component.css
  2. Edit: resources/css/index.css
  3. Add: @import "components/my-component.css";

Add new animation:
  1. Add to: resources/css/animations/general.css
  2. Use class in HTML/Blade
  3. Reference in README.md

Modify existing component:
  1. Find component file in components/
  2. Update relevant classes
  3. Test in browser
  4. Update README if needed

Change brand color:
  1. Search: #c41e3a (all references)
  2. Update in all component files
  3. Update in utilities/helpers.css
  4. Update in index.css variables
