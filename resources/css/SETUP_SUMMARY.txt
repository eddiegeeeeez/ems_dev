## How to Use

### Update Your HTML/Blade Template

Replace the CSS import in your layout file (layouts/app.blade.php):

```html
<!-- OLD -->
<link href="{{ asset('css/app.css') }}" rel="stylesheet">

<!-- NEW -->
<link href="{{ asset('css/index.css') }}" rel="stylesheet">
```
